# Medewerkers (Employees)

## Overview

Beheer van medewerkers met hun rollen, vaardigheden en beschikbaarheid.

## Components Provided

- `EmployeeList.tsx` — Grid/list view with filters
- `EmployeeDetail.tsx` — Detail with Planning and Availability tabs

## Callback Props

| Callback | Description |
|----------|-------------|
| `onEmployeeSelect` | Navigate to detail |
| `onAddEmployee` | Open creation form |
| `onEditEmployee` | Open edit form |
| `onAddAvailability` | Add availability record |
