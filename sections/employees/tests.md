# Test Instructions: Medewerkers

## Key User Flows to Test

### Flow 1: Browse Employees
- Grid view shows employee cards
- List view shows table
- Search filters by name
- Role filter works

### Flow 2: Employee Detail
- Shows info card with contact
- Planning tab shows assignments
- Availability tab shows records

### Flow 3: Availability
- Add availability record
- Shows status (ziek/vakantie/vrij)
- Edit/delete records

## Empty State Tests

- No employees → "Geen medewerkers gevonden"
- No assignments → empty planning tab
- No availability → empty list
